# SF Microclimates Skill

Get hyperlocal SF weather data.

## Triggers

- Any question about SF/San Francisco weather
- "weather tomorrow" (assume SF)
- "weather in [neighborhood]"
- "sf weather mission vs sunset"
- "is it foggy in the richmond?"
- "what's the weather like"
- Compare neighborhoods for temperature/fog

## Usage

```bash
curl https://microclimates.solofounders.com/sf-weather/{neighborhood}
```

## Neighborhoods

mission, castro, marina, soma, haight, noe_valley,
outer_sunset, inner_sunset, outer_richmond, presidio,
north_beach, pacific_heights, potrero_hill, twin_peaks,
bernal_heights, dogpatch, hayes_valley, cole_valley
